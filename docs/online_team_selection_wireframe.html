📐 设计说明

操作流程

① 选队伍
左/右 → HOME/AWAY → ② 投票选国家
方向键 + 确认 → ③ 选球场位置
上/下切换 slot → ④ READY
全员就绪开赛

人数适配

  • 2人: 每队 1 人控制, 其余 CPU
  • 4人: 每队 2 人控制 (FW×2)
  • 10人: 每队 5 人控制 (GK 始终 CPU)
  • 空位自动填充 CPU 球员
  • slot 对应 squad 中的 index (0=GK, 1-2=DEF, 3=MF, 4-5=FW)

国旗投票机制

  • 同队玩家各自投票选国家
  • 票数最多的国家为该队国旗
  • 平票时取先投的
  • 国旗右上角显示投票数 badge
  • 两队不能选同一国家

球场阵型预览

  • 中间区域实时显示两队 6 个位置
  • 已占位: 金色高亮 + 显示玩家 ID
  • 自己: 绿色脉冲动画
  • 空位: 灰色, 标注角色 (GK/DEF/MF/FW)
  • GK 位置锁定为 CPU, 不可选
  • 位置布局参考 actors_container 的 spawn 点
